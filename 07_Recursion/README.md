# Recursions in C

This folder contains Recursion Programs in C implemented to understand recursive function calls, base cases, recursive logic, and returning values.

## Programs Included:
- Factorial
- Print 1 to N Natural Numbers
- Print N to 1 Natrual Numbers
- Sum of N Natural Numbers
- Power of a Number

### Concepts Used:
- Base case
- Recursive call
- Head Recursion
- Tail Recursion
- Returning values from Recursion
- Recursion problem solving
